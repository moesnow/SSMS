# SSMS

学生学籍管理系统

## 编译

```bash
git clone https://github.com/moesnow/SSMS.git
cd SSMS
make
```

## 用法

```bash
➜  ~ ./ssms

   [学生学籍管理系统主菜单]
    +--------------------+
    |    1-添加记录     |
    |    2-修改记录     |
    |    3-删除记录     |
    |    4-查询记录     |
    |    5-统计记录     |
    |    6-排序记录     |
    |    0-退出         |
    +--------------------+

请选择(0~6):
```
